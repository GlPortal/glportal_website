    Ever Evolving, Community Driven, Free

    GlPortal is a free engine and computer game that is developed in colaboration with developers, artists and fans from all over the world.

    We blend.

    We use Blender for all of our assets and our map editor is a Blender plugin. If you are a Blender enthusiast, contact us to join our team.

    GlPortal is a free 3d action adventure

    Press download to get a build from indiedb or use the source button to get to our repository on github. If you encounter a bug, please report it.

    Download Source Code Help Report a Bug

    GlPortal is a free as in freedom game and game engine for GNU/Linux, Windows and Mac OS X.

    The player has to solve puzzles with the help of teleportation and physics manipulation.
