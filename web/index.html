    Ever Evolving, Community Driven, Free

    GlPortal is a free engine and computer game that is developed in colaboration with developers, artists and fans from all over the world.

    We collaborate.

    We are from all over the globe from a lot of different trades working together on this joint venture.

    We use blender

    for all of our assets and the GlPortal map editor is a Blender plugin. If you are a Blender enthusiast, contact us to join our team.

    Download 0.0.7.1 Source Code Join the Team Report a Bug

    GlPortal is a free as in freedom game and game engine for GNU/Linux, Windows and Mac OS X. The player has to solve puzzles with the help of teleportation and physics manipulation.

    Features

    • Modern OpenGL implementation
    • Free and open game and engine
    • Enthusiastic community
    • Active Developers
    • Challenging Tasks
